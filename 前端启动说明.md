# 前端启动说明

## ✅ 已完成的优化

## 🚀 启动前端服务

### 步骤 1：安装依赖（如果还没安装）

```bash
cd frontend
npm install
```

### 步骤 2：启动开发服务器

```bash
npm run dev
```

前端服务将运行在：**http://localhost:5173**

## 📍 访问地址

启动后，在浏览器中访问：

- **首页（商品浏览）**：http://localhost:5173
- **商品列表**：http://localhost:5173/products
- **购物车**：http://localhost:5173/cart
- **订单查询**：http://localhost:5173/order/query

## 🎯 功能说明

### 首页功能
- 显示所有商品
- 每个商品卡片显示：
  - 商品名称
  - 商品副标题
  - 价格（原价划线显示）
  - 库存和销量
  - 操作按钮：查看详情、加入购物车

### 导航栏功能
- **首页**：浏览所有商品
- **商品列表**：商品列表页面（支持搜索和筛选）
- **购物车**：查看购物车（显示商品数量）
- **订单查询**：通过订单号和手机号查询订单

## ⚠️ 注意事项

1. **后端服务必须启动**
   - 确保后端服务运行在：http://localhost:8000
   - 如果后端未启动，商品无法加载

2. **API 代理配置**
   - 前端已配置代理到后端
   - API 请求会自动转发到 http://localhost:8000

3. **浏览器控制台**
   - 如果商品加载失败，查看浏览器控制台的错误信息
   - 检查后端服务是否正常运行

## 🔧 常见问题

### 问题 1：页面显示空白
- 检查后端服务是否启动
- 检查浏览器控制台是否有错误
- 确认 API 地址配置正确

### 问题 2：商品加载失败
- 检查后端服务：http://localhost:8000/health
- 检查数据库是否有商品数据
- 查看浏览器网络请求是否成功

### 问题 3：依赖安装失败
```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```
