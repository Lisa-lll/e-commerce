## 🔍 查看数据

### 方式 1：Django Admin

1. 访问：http://localhost:8000/admin
2. 使用账户 `admin` / `admin123` 登录
3. 在"商品"和"商品分类"中查看数据

### 方式 2：前端页面

1. 启动前端服务：`cd frontend && npm run dev`
2. 访问：http://localhost:5173
3. 查看商品列表和详情

### 方式 3：API 接口

1. 商品列表：http://localhost:8000/api/v1/products/products/
2. 分类列表：http://localhost:8000/api/v1/products/categories/

## 🛠️ 管理数据

### 添加更多商品

可以通过 Django Admin 添加：
1. 访问 http://localhost:8000/admin
2. 进入"商品" → "添加商品"
3. 填写商品信息并保存

### 修改商品信息

1. 在 Django Admin 中找到要修改的商品
2. 点击商品名称进入编辑页面
3. 修改信息后保存

### 删除测试数据

如果需要清空测试数据：

```python
# 使用 Django Shell
python manage.py shell

# 在 shell 中执行
from products.models import Product, Category
Product.objects.all().delete()
Category.objects.all().delete()
```
